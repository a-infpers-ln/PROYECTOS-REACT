## Iniciar proyecto
En el archivo package.json - Darle al modo debug o ejecutar npm run dev

## Crear deployment del Proyecto
Ejecutar: npm run build (Comprobar que no hay errores)
Ir a la web: https://app.netlify.com/ - Completar formulario
En sites arrastramos la carpeta dist creada anteriormente y la pegamos dentro de la web
En el botón : 'Open production deploy' nos abrirá nuestro proyecto en la web
